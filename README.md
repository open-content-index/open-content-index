# open-content-index

Open Content Index

Looking for something to watch, listen to, read, or buy? Why should you run the gauntlet of The Algorithm? Take control of your attention.

## For End Users

Avoid Curation Scams. It is common knowledge that pay-to-play pits the needs of those with access to large amounts of money against the needs of end users trying to find, enjoy, and purchase the best content for them.

## For Creators, Record Labels, etc.

Reclaim the data you put on platforms like YouTube, SoundCloud, and Beatport. Your users should not be at the mercy of Big Tech to be able to find, enjoy, and purchase your content.

## Technical FAQ

### One giant Git repository? Are you mad?

The data can be trivially ported to (and from) more optimal (and more complicated) schemes as the need arises and as we learn more about the problem space.

### Why not use Web Scraping, API access, or AI?

We may well use these techniques to help with this effort. One issue requiring careful consideration is where the boundary of legality is drawn between "users putting links to YouTube content in a forum post" and "complete replication of all copyrighted data". This project is about [Fair Use](https://en.wikipedia.org/wiki/Fair_use).
